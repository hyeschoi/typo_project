마디모(MADYMO Models) 는 교통사고가 발생했을 때 국과수가 해당 사고로 인한 인체 상해 정도를 감정하는 말하자면 차량의 사고와 부상사이의 인과관계를 밝혀내는 가상프로그램이다. 이는 자돔차소고로 인한 부상자의 절반 정도가 경상환자임에도 입원율은 80%이상으로 이는 차양 접촉사고를 반복적으로 유도해 보험금을 타내는 등의 보험사기수단으로 전락하여 사회문제로까지 자리잡게 되자 도입된 제도이다. 마디모가 도입된지는 5년 정도가 되었지만 최근 1~2년사이 활용도가 급격히 높아졌다고 하지만 아직도 모르는 사람이 많은 것 같다. 마디모 절차는 경찰이 블랙박스 동영상 차랑파솜 흔적 도로상태 등 아고 관련자료를 첨부해서 마디몰 분석을 의뢰하면 국과수에서는 받은 자료를 마디모프로그램에 입력 마디모 분석 결과와 사고분석 내용과 함께 충돌 당시 차랑 탑승자 모습이 3D 영상으로 재현 마디모 분석 결과를 토대로 소견서를 작성한 후 경찰에 송부하면 경찰은 소견서를 바탕으로 적용여부를 결정한다. 6월 현충일 연휴에 강원도 가는길에 교통정체가 심해 차가 정체되어있는 상태에서 브레이크를 밟은채 그동안 눈에 거슬렸던 충전기선을 정리하다 브레이크를 느슨하게 밟아 그만 앞차를 받았다. 운젖석 조수석에 앉았던 부부가 목을 잡고 나왔고 차안에도 한명 더 타고 있었다. 타고내고 연락처를 주었는데 면허증 달라해서 안가지고 다닌다고 하며 연락처있고 차량번호도 있는데 문제될일이 없지 않느냐니까 주민듣록증까지 요구하는 핍해자. 결국은 사고다 보험사에접새하고 파손정도 사진도 찍고 헤어졌다.(사실 기존 흠집외에는 찾기 힘들 정도 ) 휴일이 지나고 아니나 다를까 범퍼도색에 차량렌트까지 게다가 하루가 더 지나고나니 3명 모두 진단을 받는다고 한다. (보험사가 다해줄 일이지만 피해자가 넘 지나치고 치사하다는 생각이. ) 이미 마디모를 알고 있었는데 보험사에서도 안내를 해주었다. 전화받은 직원은 댄인 담당자에게 사겨 경위와함께 마딘모 의뢰까지 이야기 해주겠다 했고 나는 전화온 대인 담당자에게 사진을 보내주었다. 원래는 마니모건의 경우는 대인접수할때 경찰서에 신고 해야한다는데 대긴 담당자가 이야기가 없어 나는 신고를 안한 상태이다. 내일 다시 연락해봐야 겠다. 마딥모가 한달정도 걸린다고 하는데 인과관계가 없을 경우 병원비와 보험금을 물어내야하고 형사처벌도 받을수 있다고 한다. 아쉬운 것은 마디모는 대인사고에만 해당되는 것 같다. 대묵의 경우는 경미한 흠집이라도 원상복구 의무가 있기 때문이라고 한다. 그래도 무조건 입원하고 보자는 나이롱 환자로 인한 대익 보상비는 많이 줄어들 것 같다. 이번삼고가 어떻게 처리될지 궁금하다. 아시는분도 계셨겠지만 도움이 되셨기를. 후기도 올리겠습니다
