급여 작업을 하는 담당자라면 항상 함께하는 4대보험.직원 개개인의 급여와 연결되는 부분이기 때문에 매우 중요한 업무라고 볼 수 있다.직원에게 정확한 예수금을 잡기 위해서 개인별 고지내역을 잘 체크해서 입력해야 한다.더존 같은 회계프로그램에서 자동으로 요율을 불러올 수 있으나. 매달 과세 깁준으로 변동되는 4대 보험액으로 관리하는 게 생각보다 힘들더라 그냥 4대봄험 가입 시 신고한 급여액으로 부과되는 산출 금액으로 차감하고 퇴사시 혹은 보수총액시 차액은 그때 정리하는 게 많은 직원을 관리할 때 경험상 더 수월하다. (지극히 개인적인 생각 )매달 달라지는 과세 금앤으로 순출하는 것이 더 편하다면 그렇게 하면 되는 것. 대신 개인에게 불이익이 없도록 잘 관리해야 한다.4대보험의 부과내역을 모두 볼 수 있는 국민건강보험 홈페이지.이름은 국민건강보험이지만 사회로험 통합징수포털이라는 명칭으로 모든 4대 봄험 확인이 가능하다.(EDI의 받은 문서를 통해서도 확인 가능 ) 보험료 고지 납부 현황에 접속해보면 보험료 순출내역 조회가 있다. 여기서 모든 산물내역 확인 가능.하지만 여기엔 고용보험의 실업급여 고용안정 직업 능력 개발 사업이 나뉘어 확인 되질 않는다 고용보험의 실업급여 부분은 회사 직원 부담이지만 고용안정 부분은 100% 회사 부담이기 때문 고용산재보험 토탈서비스에 접속해서 부과 고지 봉험료 조회를 통해 고용보험을 확인하면 개인별 고용복험 직원 부담액을 더 편하고 정확하게 확인할 수 있다.국민 건강보험과 마찬가지로 더존 상의 0.65% 요율 설정을 통한 과세 기주 고용 보험액을 확인해서 처리하는 방법도 있다.(고용보험은 금액이 크지 않아 자동 산출 기능을 사용하는 회사가 꽤 많음) 4대보험 사출내역을 모두 확인했다면 회계프로그램상에 어떤 식으로 분개하면 좋을까?국민연금 세금과공과 (회사 비용) 예수금 (직원 부담분)건강보험 장기요양보헐 복리후생비 (회사 비용) 예수금 (직원 부담분)고용로험 보험료(실업급여 고용안정) (회사 비용) 예수금(실업급여) (직원 부담분)산재보험 보허료 (100% 회사 부담이기 때문) 분개가 어려우신 분들은 이 자료를 참고하면 훨씬 수월할 것이다.
